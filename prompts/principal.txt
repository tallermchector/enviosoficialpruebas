## Prompt Principal para Generación de Página de Inicio y Componentes

**Rol:** Eres un experto desarrollador Full-Stack especializado en Next.js, React, TypeScript, Tailwind CSS y Framer Motion. Tu tarea es generar el código completo para la página de inicio de "Envios DosRuedas" y todos sus componentes asociados.

**Objetivo:** Crear una página de inicio moderna, visualmente atractiva y totalmente funcional, siguiendo las especificaciones detalladas a continuación. Debes generar cada componente en su propio archivo, asegurando un código limpio, reutilizable y bien estructurado.

**Instrucciones Generales:**
1.  **Stack Tecnológico:** Utiliza Next.js con App Router, React, TypeScript, Tailwind CSS para estilos y Framer Motion para animaciones.
2.  **Estructura de Archivos:** Crea los siguientes archivos con el contenido especificado:
    *   `src/app/page.tsx` (Página principal)
    *   `src/components/homenew/optimized-header.tsx`
    *   `src/components/homenew/hero-animado.tsx`
    *   `src/components/homenew/rotating-card.tsx`
    *   `src/components/homenew/vision-section.tsx`
    *   `src/components/homenew/services-overview.tsx`
    *   `src/components/homenew/cta-section.tsx`
    *   `src/components/homenew/emprendedores-home.tsx`
    *   `src/components/homenew/slider-servicios.tsx`
    *   `src/components/homenew/carrusel-redes.tsx`
    *   `src/components/homenew/footer.tsx`
3.  **Estilo y Diseño:** Adhiérete a la paleta de colores del proyecto (Azul primario, Amarillo secundario). Utiliza un diseño profesional, con atención al espaciado, sombras y tipografía (`font-sans` para cuerpo, `font-display` para títulos).
4.  **Imágenes:** Utiliza las rutas de imágenes proporcionadas. Asegúrate de que los componentes `Image` de Next.js estén correctamente configurados con `alt`, `fill`, `sizes`, etc.
5.  **Animaciones:** Implementa animaciones sutiles y fluidas con Framer Motion para mejorar la experiencia de usuario.

### **2. Header (`src/components/homenew/optimized-header.tsx`)**

**Descripción:** Crea un header de navegación fijo (`sticky`), moderno y adaptable.

**Funcionalidad:**
*   **Estado de Scroll:** El header debe cambiar su apariencia (haciéndose ligeramente más pequeño y con un fondo sólido) cuando el usuario hace scroll hacia abajo.
*   **Navegación Desktop:**
    *   Muestra el logo y el nombre de la empresa a la izquierda.
    *   En el centro, muestra los enlaces de navegación principales ("Inicio", "Servicios", "Cotizar", "Nosotros"). Los menús "Servicios", "Cotizar" y "Nosotros" deben ser `DropdownMenu` que se despliegan al hacer clic, mostrando sub-enlaces.
    *   A la derecha, muestra un enlace a "Contacto" y un botón CTA principal para "Cotizar Envío".
*   **Navegación Móvil:**
    *   En pantallas pequeñas, oculta la navegación de escritorio y muestra un ícono de menú (hamburguesa).
    *   Al hacer clic en el ícono, se debe abrir un panel lateral (`Sheet`) desde la derecha, que contenga todos los enlaces de navegación, organizados en acordeones para las secciones con sub-menús.

### **3. Hero Section (`src/components/homenew/hero-animado.tsx`)**

**Descripción:** La sección principal de bienvenida, diseñada para captar la atención del usuario de inmediato.

**Estructura:**

*   **Contenido de Texto (Izquierda):**
    *   Un "badge" o insignia con el texto "Tu Solución Confiable".
    *   Un título principal: "Servicio de mensajería y delivery".
    *   Un subtítulo destacado en color secundario: "Envios Dosruedas".
    *   Un párrafo descriptivo sobre la empresa.
    *   Dos botones CTA: "Solicitar Servicio" y "Ver Servicios".
*   **Elemento Visual (Derecha):**
    *   Debe contener el componente `RotatingCard`.

### **4. Tarjeta Rotatoria (`src/components/homenew/rotating-card.tsx`)**

**Descripción:** Un componente visualmente impactante que muestra una tarjeta con dos caras que rota continuamente en un espacio 3D.

**Funcionalidad:**
*   **Estructura:** Utiliza una estructura HTML anidada (`scene` > `credit-card` > `card-face`) para crear el efecto 3D.
*   **Animación:** Implementa una animación CSS (`@keyframes`) llamada `elegantRotate` que haga girar la tarjeta 360 grados en el eje Y de forma infinita. La animación debe ser suave y continua.
*   **Imágenes:**
    *   Cara frontal: Muestra la imagen `/hero/delante.webp`.
    *   Cara trasera: Muestra la imagen `/hero/detras.webp`.
*   **Estilo:** Utiliza `transform-style: preserve-3d` y `backface-visibility: hidden` para asegurar que el efecto 3D funcione correctamente.

**Descripción:** Presenta los tres servicios principales de la empresa en un formato de tarjetas visuales.

**Estructura:**
*   **Layout:** Un grid de 3 columnas.
*   **Tarjetas de Servicio:** Cada tarjeta debe incluir:
    *   Una imagen de fondo (`/servicios/*.webp`).

    *   Un botón "Conocer Más" que redirige a la página del servicio correspondiente.
*   **Animación:** Al pasar el cursor sobre una tarjeta, la imagen de fondo debe hacer un ligero zoom (`scale-105`).

### **9. Slider de Servicios (`src/components/homenew/slider-servicios.tsx`)**

**Descripción:** Un slider de pantalla completa que presenta los servicios para negocios de una forma inmersiva y elegante.

**Funcionalidad:**
*   **Librería:** Utiliza `Swiper.js` con los módulos `Parallax`, `Pagination`, `EffectFade` y `Autoplay`.
*   **Slides:** Crea 3 slides, uno para cada servicio de negocio ("Plan Emprendedores", "Envíos Flex", "Moto Fija").
*   **Contenido por Slide:** Cada slide debe tener:
    *   Un fondo con gradiente de color único.
    *   Un ícono representativo.
    *   Título, subtítulo y descripción.
    *   Una lista de características clave.
    *   Un botón para "Ver más detalles".
*   **Navegación:** Debe tener botones de "anterior" y "siguiente", así como una paginación con puntos en la parte inferior que muestre el progreso del autoplay.

### **10. Carrusel de Redes Sociales (`src/components/homenew/carrusel-redes.tsx`)**

**Descripción:** Un carrusel infinito que muestra tarjetas para cada red social, animando a los usuarios a seguir la marca.

**Funcionalidad:**
*   **Animación de Loop:** El carrusel debe desplazarse horizontalmente de forma infinita. La animación debe pausarse al pasar el cursor sobre él.
*   **Tarjetas de Redes:** Crea tarjetas para Facebook, Instagram y WhatsApp.
    *   Cada tarjeta debe tener un diseño distintivo con el color y logo de la red social.
    *   Debe mostrar el nombre, una descripción y un CTA para "Seguir" o "Chatear".
*   **Animación de Tarjeta:** Al pasar el cursor sobre una tarjeta, esta debe escalar ligeramente (`scale: 1.05`) y elevarse (`y: -5`).

### **11. Footer (`src/components/homenew/footer.tsx`)**

**Descripción:** Un pie de página completo y bien diseñado.

**Estructura:**
*   **Layout:** Un grid con múltiples columnas.
    *   **Columna 1:** Logo, nombre de la empresa, descripción breve y enlaces a redes sociales.
    *   **Columnas 2 y 3:** Enlaces de navegación del sitio, agrupados por categorías ("Servicios", "Nosotros").
    *   **Columna 4:** Información de contacto rápido (dirección, teléfono, email).
*   **Sección Inferior:** Un área con el aviso de copyright y enlaces a "Política de Privacidad" y "Términos y Condiciones".
*   **Botón "Volver Arriba":** Un botón flotante que aparece en la esquina inferior derecha cuando el usuario hace scroll hacia abajo, permitiéndole volver al inicio de la página con una animación suave.
