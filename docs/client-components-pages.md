# Páginas que son Client Components

A continuación se listan todos los archivos `page.tsx` que funcionan como **Client Components**. Estos componentes están marcados con la directiva `"use client";` al principio del archivo.

Son necesarios cuando la página requiere interactividad del lado del cliente, como el uso de **hooks** (`useState`, `useEffect`, etc.) o la gestión de **eventos** (`onClick`, `onChange`).

- `/src/app/admin/etiquetas/page.tsx`
- `/src/app/seguimiento/[id]/page.tsx`
- `/src/app/seguimiento/destinatarios/page.tsx`
- `/src/app/seguimiento/etiquetaslote/page.tsx`
- `/src/app/seguimiento/page.tsx`
- `/src/app/seguimiento/perfil/page.tsx`
