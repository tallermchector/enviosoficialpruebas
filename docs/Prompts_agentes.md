# 🚀 Guía Maestra para Prompts en Stitch

Para obtener diseños de alta fidelidad, consistentes y profesionales desde el primer intento, estructura tus peticiones siguiendo este esquema. Stitch procesa mejor los prompts que definen **Contexto**, **Estructura**, **Estilo** y **Contenido**.

---

## 📋 Estructura de un Prompt "Perfecto"

Copia y completa esta plantilla para tus solicitudes:

### 1. El Objetivo (¿Qué estamos construyendo?)
> "Crea una [Tipo de Pantalla: Landing Page / Dashboard / Formulario] para [Nombre del Proyecto/Marca]."

### 2. Contexto y Audiencia (¿Para quién es?)
> "El objetivo es [Propósito: vender, informar, registrar] dirigido a [Audiencia: usuarios finales, empresas, perfiles técnicos]."

### 3. Estructura y Secciones (¿Qué debe contener?)
> "La página debe incluir las siguientes secciones en orden:
> - **Hero:** [Imagen de fondo / Título impactante / CTA principal]
> - **Beneficios:** [Lista de 3-4 puntos clave]
> - **Prueba Social:** [Logos de clientes / Testimonios]
> - **Acción Final:** [Formulario de contacto / Botón de compra]"

### 4. Identidad Visual (¿Cómo debe verse?)
> "Aplica el siguiente estilo:
> - **Colores:** [Primario, Secundario, Acentos]
> - **Tipografía:** [Fuentes específicas o estilo: 'Moderno sin-serif', 'Clásico con serifa']
> - **Estética:** [Minimalista, Tech/Futurista, Corporativa, Colorida]"

### 5. Restricciones Negativas (¿Qué NO queremos?)
> "Evita el uso de [ej: bordes redondeados, colores rojos, imágenes genéricas de stock]."

## 💡 Consejos de Experto para Stitch

*   **Verbatim:** Si tienes un eslogan o nombre de marca exacto, escríbelo entre comillas: `"Envíos DosRuedas"`. Lo respetaré letra por letra.
*   **Referencias:** Si te gusta una imagen o pantalla que ya está en el canvas, selecciónala y dime: *"Usa el estilo de esta pantalla pero cambia el contenido a..."*
*   **Iteración:** No intentes hacerlo todo en un solo prompt gigante. Es mejor crear la estructura base y luego decir: *"Ahora añade efectos de cristal (glassmorphism) a las tarjetas."*
*   **Dispositivo:** Siempre menciona si lo quieres para `DESKTOP` o `MOBILE`.
