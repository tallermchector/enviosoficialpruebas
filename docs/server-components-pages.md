# Páginas que son Server Components

A continuación se listan todos los archivos `page.tsx` del proyecto que funcionan como **Server Components**. Estos componentes se ejecutan principalmente en el servidor, son ideales para la obtención de datos (`async/await`) y no envían JavaScript innecesario al cliente, mejorando el rendimiento.

Por defecto, todas las páginas en el App Router de Next.js son Server Components a menos que se especifique lo contrario con la directiva `"use client";`.
